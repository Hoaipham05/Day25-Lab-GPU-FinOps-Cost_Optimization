# Submission Template

Dung thu muc nay lam khung bai nop.

## Cau truc

```text
submission_template/
|- CHECKLIST.md
|- report.pdf
|- screenshots/
|- generated_charts/
|- notebook/
```

## Ghi chu

- `report.pdf` la tuy chon neu mon hoc yeu cau.
- Cac thu muc con da co README huong dan ten file can dat vao.
- Doi ten `submission_template` thanh ten thu muc nop bai theo mau giang vien yeu cau khi da hoan tat.
